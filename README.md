# picow-ota
over the air updates proof of concept
https://github.com/kevinmcaleer/ota
1. Load Kevin's `ota.py` onto your Pico W
2. Also create a `WIFI_CONFIG.py` file (so you can load your wifi credentials)
3. Ensure you've the correct GitHub repo referenced in your program that drives the update
4. Ensure you've the proper main
5. Update your main.py
6. Update the version.json file (this is what drives the OTA program to know to pull down new code)
